# 🚐 Conductor App — Firebase Edition

App de gestión diaria para conductor de flota (Uber/Cabify) en Bogotá.
Conectada a Firebase Firestore para sincronización en tiempo real.

## Funcionalidades
- Registro de ganancias diarias con distribución 60/40
- Gastos (gas natural, gasolina, varios) con foto de recibo comprimida
- Control de turnos
- Viajes personales fuera de plataforma
- Pico y Placa (placa QON389)
- Exportar reporte Excel (.xlsx)
- **Sincronización en tiempo real con Firebase**

## Despliegue
1. Sube a GitHub
2. Importa en Vercel → Deploy
